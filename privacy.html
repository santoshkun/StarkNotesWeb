Privacy Policy

Last updated: July 15, 2026

StarkNotes is a browser extension for capturing and organizing notes from webpages. This page explains exactly what data the extension handles and where it goes.

What's stored, and where

All notes, tags, and settings you create in StarkNotes are stored locally on your device using Chrome's built-in storage APIs. Nothing you write in StarkNotes — no note content, no page text, no highlights — ever leaves your browser or is sent to any server operated by StarkNotes.

The one external request StarkNotes makes

To show a small site icon next to each note, StarkNotes requests a favicon from Google's public favicon service, using only the domain name of the site the note was taken on (for example, example.com — never the full page URL, page title, or note content).

This is a display feature only. It does not transmit anything you've written, and it isn't used for tracking or analytics.

What StarkNotes does not do

  • Does not send note content, page text, or browsing history to any server
  • Does not use analytics, trackers, or advertising SDKs
  • Does not sell or share data with third parties
  • Does not require an account or login

Permissions explained

  • Storage — to save your notes locally on your device
  • Active tab / scripting — to detect selected text on the current page so it can be captured as a note
  • Tabs — to identify the current page's title and URL when creating a note, and to open StarkNotes-related pages (like the shortcuts settings page)

Changes to this policy

If this policy changes, the update date above will change accordingly. Continued use of StarkNotes after an update means you accept the revised policy.
