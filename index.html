Features

Built for how researchers actually work

Not a bookmark manager. Not a read-later app. A fast capture layer that lives between the page and your thinking.

Instant capture
Select text, press Alt+Shift+S. Title, URL, favicon, and timestamp are auto-filled. No forms, no friction.
background.js → chrome.storage.local
Append mode
Alt+Shift+F appends a timestamped bullet · [14:20] text to today's note for that site. One rolling log per domain per day.
aggregate notes · no duplicates
Inline hashtags
Write #trading anywhere in a note — it's automatically extracted as a tag and indexed for filtering. Hashtags stay in your text.
auto-extracted · non-destructive
Browse by site or tag
Sidebar shows every domain and tag you've ever captured from, sorted by count. Tap any to filter the list. Tap a section header to scope to that site instantly.
sidebar · filter bar · one-click clear
Pinned notes
Pin any note from the detail view. Pinned notes surface in a collapsible panel below search for quick access while staying visible in the main list with a pin indicator.
quick access · never hidden
Export & import
Export all notes or a selection as .md or .json. Import back with deduplication by ID and title+URL fingerprint — reimporting is always safe.
portable · fingerprint deduplication
How it works

Three keystrokes cover 90% of your workflow

StarkNotes is designed around muscle memory. You don't think about it — you just capture, and it's there.

01
Select and capture
Highlight any text on any page. Press Alt+Shift+S. A green toast confirms the save. The selection is highlighted on the page. Done.
02
Append as you go
Keep reading. When you hit something worth adding to the same note, press Alt+Shift+F. It appends a timestamped bullet to today's log for that site — no new note created.
03
Surface and review
Open the popup with Alt+Shift+M. Your notes are grouped by site, filtered by tag, searchable, sortable, and exportable — all without leaving the browser.
Shortcuts

Every action has a key

Alt+ Shift+ S
Capture or new note
Selection → saves immediately. Nothing selected → opens blank editor pre-filled with page title.
Alt+ Shift+ F
Append to today's note
Adds a timestamped bullet to the current site's daily log. Creates one if it doesn't exist yet.
Alt+ Shift+ M
Open popup
Opens StarkNotes from anywhere, any page, without clicking the toolbar icon.
Ctrl+ ↵
Save note
Save from any field in the editor — title, content, or tag input.
Esc
Save and go back
Saves the current editor state and navigates back. Clicking away also auto-saves.
↑ ↓
Navigate notes
Move between cards in the list. Enter to open, Backspace to go back.
Takes about 60 seconds
